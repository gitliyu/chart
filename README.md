# chat
基于vue，使用socket.io实现的移动端实时通信
### 执行命令
```
npm install
npm run start
```
使用模拟数据登录，用户信息见`src/assets/js/users.js`<br>

纯前端版本，完整版请移步['chat-Laravel'](https://github.com/gitliyu/chat-Laravel) (In progress)
